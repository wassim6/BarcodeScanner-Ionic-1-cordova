# BarcodeScanner Ionic 1 cordova
A simple BarcodeScanner with Ionic 1, angularJs and  plugin BarcodeScanner which has the ability to scan barcodes in the following formats:  
<ul>
<li>QR Code </li>
<li>Data Matrix </li>
<li>UPC E </li>
<li>UPC A </li>
<li>EAN 8 </li>
<li>EAN 13 </li>
<li>Code 128 </li>
<li>Code 39 </li>
<li>ITF</li>
</ul>
